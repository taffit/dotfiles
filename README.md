# dotfiles
This repository contains my personal dotfiles.

After having to fiddle on too many systems, *The Big Purge* (or: *Großes Reinemachen*) happened on the repository. Instead of archiving the repository and recreating a new one, I decided to move everything to a temporary folder which will then get deleted. The new dotfiles will be recreated step by step using [Nix][https://nixos.org/].

A huge journey ahead of me, starting with this first step.
